# deep-isalaniya.is-a.dev

Personal portfolio site — built with vanilla HTML/CSS/JS, hosted on GitHub Pages.

## What's inside

Single-page portfolio covering my work in AI/ML — computer vision, speech AI, and NLP projects. No frameworks, no build step, just straight HTML.

## Stack

- Vanilla HTML / CSS / JS
- Google Fonts (Syne + JetBrains Mono)
- Hosted via GitHub Pages + is-a.dev subdomain

## Local preview

Just open `index.html` in a browser. No setup needed.
