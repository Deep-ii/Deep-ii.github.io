Portfolio

Urban City Analysis
A multi-module machine learning project focused on building smart city solutions through data analytics. It covers three major problem areas:
  • Crime Prediction: Stacking Classifier, Random Forest, and Logistic Regression used to predict the likelihood of crimes based on location, time, and other city factors.
  • Accident Severity: Decision Tree, SVM, and Logistic Regression models classify road accident severity using weather, environmental, and visibility data.
  • Passenger Count Prediction: Linear Regression model estimates public transport passenger counts using time-based and route features.

Tools used: Python, Scikit-learn, Pandas, Matplotlib
